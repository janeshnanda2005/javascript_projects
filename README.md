# JavaScript Projects

A collection of JavaScript projects built with vanilla JS, HTML, CSS, and Node.js.

🔗 **Live Demo:** [javascript-projects-murex-rho.vercel.app](https://javascript-projects-murex-rho.vercel.app)

## Projects

### 📝 NotesApp

A browser-based notes application that lets users create, view, and manage notes.

**Tech Stack:** HTML · CSS · JavaScript

### 🌐 Node (Server-side App)

A server-rendered application using Node.js and EJS templating.

**Tech Stack:** Node.js · EJS · JavaScript

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)

### Run locally

```bash
# Clone the repository
git clone https://github.com/janeshnanda2005/javascript_projects.git
cd javascript_projects
```

**For the NotesApp** (open directly in browser):
```bash
open NotesApp/index.html
```

**For the Node app:**
```bash
cd node
npm install
node index.js
```

Then open `http://localhost:3000` in your browser.
